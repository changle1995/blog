# Java多人博客系统

### 涉及技术：

* JDK8
* MySQL
* Spring-boot
* Spring-data-jpa
* Spring-security
* Swagger2

### 启动：
 - main方法运行
 ```
 配置：src/main/resources/application-datasource.yml (数据库账号密码)、向数据库导入初始数据
 运行：src/main/java/com/example/blog/BootApplication
 访问：http://localhost:10080/
 账号：默认管理员账号为 account/12345
```
